# 🌱 Can't Touch Ground
*A Minecraft datapack that instantly kills players when they touch ground.*

## 📖 Overview
If a player stands on any of natural surface blocks, they die immediately.
### 💀 List of deadly blocks
- Grass
- Dirt
- Coarse dirt
- Rooted dirt
- Podzol
- Mycelium
- Sand
- Red sand
- Snow
- Ice
- Packed ice
- Blue ice
- Mud
- Farmland
- Dirt path
- Netherrack
- Soul sand
- Soul soil
- Blackstone
- Basalt
- Magma
- Crimson nylium
- Warped nylium
- Endstone

## 📦 Installation
### Singleplayer
- Download the latest release .zip file.

- Place it inside your world's datapacks/ folder.

- Run /reload in the world (or re‑enter the world).

### Server
- Upload the .zip to the datapacks/ folder of your server world, then restart or run /reload.
